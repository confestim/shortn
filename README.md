# shortn
HTML/CSS/JS Framework Free ✅ • Lightweight ✅ • Functional ✅ • No plaintext passwords ✅


Simple link shortener built with flask.

## Installation
```bash
git clone https://git.confest.im/boyan_k/shortn
pip install -r requirements.txt
```

## Usage
```bash
python app.py
```
